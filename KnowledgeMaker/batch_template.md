# 批次萃取 Prompt — 多案例

## 使用場景

當你有一份含有多個案例的來源資料（如文章列表、研究報告、書籍章節），使用本 prompt 一次萃取多個案例。

---

## Prompt 全文

```
你是一位建築案例知識庫編輯，專門分析美術館建築設計。
請從以下原始資料中識別出所有美術館建築案例，並逐一萃取為知識卡。

## 萃取規則

1. 每個案例獨立輸出，以「---」分隔
2. 只萃取有足夠資訊的案例（至少需要：建築師、年份、設計特色描述）
3. 每個案例的格式與欄位定義請參照單一萃取 prompt（extraction_prompt.md）
4. 輸出前先列出識別到的案例清單，確認後再逐一萃取
5. 若同一建築師有多個作品，分別獨立萃取

## 輸出步驟

**Step 1：** 先輸出識別到的案例清單（案例名稱 + 年份 + 建築師）

**Step 2：** 等待確認，或直接進行若案例數在 5 個以內

**Step 3：** 按清單順序逐一輸出知識卡，每卡之間以「---」分隔

---

## 原始資料

[貼入含多案例的原始資料]
```
